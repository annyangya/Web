#### box-shadow

```css
box-shadow:5px 6px 10px 2px rgba(0,0,0,0.4);

/*
对应参数意义：
5px：x方向拓展区域
6px：y方向拓展区域
10px：模糊区域的宽度，模糊区域为0就是一个边框
2px：扩展区域的宽度，拓展区域为0也是一个边框
rgba：阴影的颜色加透明程度
*/
```

- 营造层次感（立体感）
- 充当没有宽度的边框
- 特殊效果实现
- 画出各种图案

#### border-radius

- 圆角矩形
- 圆形
- 半圆/扇形
- 奇怪的角

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style type="text/css">
        .container{
            /*border: 5px solid red;*/
            width: 100px;
            height: 100px;
            background: lightpink;
            border-radius: 50%;/*50%为圆形，50%以上就没有效果了*/
            border-radius: 50px;
            border-top-left-radius: 100px;
            border-top-right-radius: 0;
            border-bottom-left-radius: 0;
            border-bottom-right-radius: 0;/*扇形*/
            border-radius: 100px 50px;
        }
    </style>
</head>
<body>
<div class="container"></div>
</body>
</html>
```

#### background

- 纹理、图案
- 渐变
- 雪碧图动画
- 背景图片尺寸适应

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style type="text/css">
        .container{}
        /*图标动态效果*/
        .i{
            width: 20px;
            height: 20px;
            background: url("background.png") no-repeat;
            background-size: 20px 40px;
            transition: background-position .4s;
        }
        .i:hover{
            background-position: 0 -20px;
        }
        .img{
            width: 400px;
            height: 300px;
            background: url("panda.jpg") no-repeat;
            border: 1px solid red;
            background-position: center top;/*图片显示不全，这里我们让图片居中显示，偏上*/
            /*background-size: 200px 200px;*/
            background-size: cover;/*让图片覆盖整个容器，长宽比不变，多的地方弃掉*/
            background-size: contain;/*图片保持不变，如果图片太小就可能会有空白*/
        }
    </style>
</head>
<body>
<div class="container">
    <div class="i"></div>
    <div class="img"></div>
</div>
</body>
</html>
```

#### clip-path

按路径进行裁剪

- 对容器进行裁剪
- 常见几何图形裁剪
- 自定义路径裁剪

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style type="text/css">
        .container{
            width: 400px;
            height: 300px;
            background: url("panda.jpg") no-repeat center;
            clip-path: inset(100px 50px);/*裁剪矩形*//*所处位置*/
            clip-path: circle(50px at 100px 100px);/*在100*100的位置有一个半径为50px的圆形*/
            clip-path: polygon(50% 0%,100% 50%,50% 100%,0% 50%,10px 10px);/*定义一个方形，也可以为不规则图形,加上后面的10px以后*/
            clip-path: url(#circle1);
        }
        .container:hover{
            clip-path: circle(80px at 100px 100px);
        }
    </style>
</head>
<body>
<div class="container"></div>
<svg>
    <defs>
        <clipPath id="circle1">
            <circle  cx="60 cy=60" r="50" fill="#34538b"></circle>
            <!--实现五角星-->
            <polygon stroke="#979797" points="0.4921875 81.2070313 92.640625 81.2070313 121.601562 0.21875 153.648437 81.2070313 247.390625 80.7734375 173.394531 140.496094 200.308594 227.09375 121.601562 172.71875 53.4960937 227.09375 80.859375 140.496094"></polygon>
        </clipPath>
    </defs>
</svg>
</body>
</html>
```

#### 3D变换

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style>
        .container{
            margin:50px;
            padding: 10px;
            border: 1px solid red;
            width: 200px;
            height: 200px;
            position: relative;
            perspective: 500px;
        }
        #cube{
            width:200px;
            height:200px;
            transform-style: preserve-3d;
            transform: translateZ(-100px);
            transition:transform .4s;
        }
        #cube div{
            width: 200px;
            height:200px;
            position: absolute;
            line-height: 200px;
            font-size:50px;
            text-align: center;
        }
        #cube:hover{
            /* transform: translateZ(-100px) rotateX(270deg); */
            transform: translateZ(-100px) rotateX(90deg) rotateY(90deg);
        }
        .front{
            transform: translateZ(100px);
            /* transform: translateX(100px); */
            /* transform: translateX(100px) translateY(10px) rotate(25deg); */
            /* transform: rotate(25deg) translateX(100px) translateY(10px); */
            background:rgba(255,0,0,.3);
        }
        .back{
            /* transform: translateZ(-100px); */
            transform: translateZ(-100px) rotateY(180deg);
            background:rgba(0,255,0,.3);
        }
        .left{
            transform: translateX(-100px) rotateY(-90deg);
            background:rgba(0,0,255,.3);
        }
        .right{
            transform: translateX(100px) rotateY(90deg);
            background:rgba(255,255,0,.3);
        }
        .top{
            transform: translateY(-100px) rotateX(-90deg);
            background:rgba(255,0,255,.3);
        }
        .bottom{
            transform: translateY(100px) rotateX(90deg);
            background:rgba(0,255,255,.3);
        }

        
    </style>
</head>
<body>
    <div class="container">
        <div id="cube">
            <div class="front">1</div>
            <div class="back">2</div>

            <div class="left">4</div>
```

### CSS面试真题

- 如何用一个div画xxx
  - box-shadow无限投影
  - ：：after
  - ：：before
- 如何产生不占空间的边框
  - box-shadow
  - outline
  - 可以设置box-size 为border-content，内容包含边框
- 如何实现圆形元素（头像）
  - border-radius：50%
- 如何实现IOS圆标的圆角
  - clip-path：（svg），倒出svg
- 如何实现半圆扇形图形
  - border-radius组合：
    - 有无边框
    - 边框粗细
    - 圆角半径
- 如何实现背景图居中显示/不重复/改变大小
  - background-position
  - background-repeat
  - background-size(cover/contain)
- 如何平移/放大一个元素
  - transform：translateX（100px）
  - transform：scale（2）
- 如何实现3D效果：
  - perspective：500px（指定透视角度）
  - transform-style：preserve-3d；（指明是3d效果）
  - transform：translate rotate；（图形的各种变换）
