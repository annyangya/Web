## float浮动

#### 浮动

float 四个参数：left, right,none,inherit

- left：左浮动
- right：右浮动
- none：不浮动
- inherit：继承父元素浮动属性

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style type="text/css">
        /*清除所有边距*/
        *{
            padding: 0;
            margin: 0;
        }
        .text{
            width: 100px;
            height: 100px;
            background-color: red;
            float: inherit;
            margin-right: 10px;
        }
        .pre{
            width: 300px;
            height: 300px;
            background-color: blue;
            float: left;
        }

        /*div换行另起一行，如果float：left则左排列成一行*/
    </style>
</head>
<body>
<div class="pre">
    <div class="text"></div>
    <div class="text"></div>

</div>

</body>
</html>
```

左浮动排列顺序从左到右，右浮动排列顺序从右到左。

- float浮动可使内联元素变为块级元素

  ```Html
  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <style type="text/css">
          .one{
              width: 100px;
              height: 100px;
              background-color: red;
              float: left;
          }
      </style>
  </head>
  <body>
  <span class="one">adsa</span>
  </body>
  </html>
  ```

如上，span标签是内联元素，不能设置width，height，无法显示，但是设置浮动之后就可以显示出来。

#### CSS 中的定位机制

1. 标准流（普通流）
2. 定位
3. 浮动

#### 浮动副作用解决

- 手动给父元素添加高度

问题：如果子元素太多，设置父元素高度可能装不下子元素，子元素就回超出父元素的高度

如：

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style type="text/css">
        *{
            margin: 0;
            padding: 0;
        }
        .text{
            width: 100px;
            height: 30px;
            background-color: red;
            border: 1px white solid;
            float: left;
        }
        .per{
            width: 500px;
            height: 32px;
            border: 1px black solid;
        }
    </style>
</head>
<body>
<div class="per">
    <div class="text"></div>
    <div class="text"></div>
    <div class="text"></div>
    <div class="text"></div>
    <div class="text"></div>
    <div class="text"></div>
    <div class="text"></div>
    <div class="text"></div>
</div>
<div class="bro"></div>


</body>
</html>
```

- 通过clear清除内部和外部浮动

  clear参数：

  - none
  - left：不允许左边有浮动对象
  - right：不允许右边有浮动对象
  - both：不允许有浮动对象

  ```
  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <style type="text/css">
          *{
              margin: 0;
              padding: 0;
          }
          .text{
              width: 100px;
              height: 30px;
              background-color: red;
              border: 1px white solid;
              float: left;
          }
          .per{
              width: 200px;
              height: 400px;
              border: 1px black solid;
          }
          .tex1{
              width: 100px;
              height: 100px;
              background-color: red;
              float: right;
              /*添加浮动后，和tex2不在一个文本流中，所以会覆盖在上面*/

          }
          .tex2{
              width: 120px;
              height: 120px;
              background-color: blue;
              clear: both;/*不允许左边有浮动对象，所以二者最终上下排列*/
          }
      </style>
  </head>
  <body>
  <div class="per">
      <div class="tex1"></div>
      <div class="tex2"></div>
  </div>
  <div class="bro"></div>
  </body>
  </html>
  ```

- 给父元素添加overfloat属性并结合zoom：1使用

  overflow：hidden/auto，处理溢出问题

  zoom：IE专用属性，可以通过子元素的大小来放大缩小父元素的大小，导致父元素拥有高度

  弊端：一旦子元素内容过多，会出现被截断问题

  ```html
  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <style type="text/css">
          *{
              margin: 0;
              padding: 0;
          }
          .text{
              width: 100px;
              height: 30px;
              background-color: red;
              border: 1px white solid;
              float: left;
          }
          .per{
              width: 500px;
              height: auto;
              border: 1px black solid;
              overflow: hidden;
              zoom: 1;
          }

      </style>
  </head>
  <body>
  <div class="per">
      <div class="text"></div>
      <div class="text"></div>
      <div class="text"></div>
      <div class="text"></div>
      <div class="text"></div>
      <div class="text"></div>
  </div>
  <div class="bro"></div>
  </body>
  </html>
  ```
